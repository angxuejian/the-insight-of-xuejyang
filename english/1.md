# 英语十大词性

| 词性  | 英文           | 核心作用        | 常见问题            |
| --- | ------------ | ----------- | --------------- |
| 名词  | Noun         | 表示“东西”      | 谁？什么？           |
| 代词  | Pronoun      | 替代名词        | 他/它/这/那些        |
| 动词  | Verb         | 表示动作/状态     | 做什么？            |
| 形容词 | Adjective    | 修饰名词        | 什么样的？           |
| 副词  | Adverb       | 修饰动作/形容词/句子 | 怎么样地？           |
| [介词](./1/6.md)  | Preposition  | 表示关系        | 在哪、朝哪、为了谁       |
| [冠词](./1/7.md)  | Article      | 限定名词        | a/an/the        |
| 连词  | Conjunction  | 连接内容        | and/but/because |
| 感叹词 | Interjection | 表达情绪        | wow/oh          |
| 数词  | Numeral      | 表示数量        | one/two/first   |
