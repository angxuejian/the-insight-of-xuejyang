# 构词法

> 通过前缀、后缀等方式，在已有单词的基础上创造新的单词，并扩展新的含义。
>
>
> 核心思想：不要背单词，要学会拆单词（Prefix + Root + Suffix）


```text
单词（Word）
│
├── 前缀（Prefix）= 修饰核心意思
├── 词根（Root） = 核心意思
└── 后缀（Suffix）= 决定单词角色（词性）
```

> 前缀负责改意思，词根负责表达核心，后缀负责决定身份。

```text
Prefix = 改意思（What）

Root = 核心意思（Core）

Suffix = 决定身份（Role）
```

```text
misunderstanding

mis + under + stand + ing

mis    = 错误地
under  = 在下面（这里属于词根的一部分）
stand  = 站立 → 理解
ing    = 名词化

= 误解
```


## Prefix 

| 前缀 | 含义 | 示例 |
|------|------|------|
| re- | 再次、重新 | reuse, rebuild, review |
| un- | 不、相反 | unhappy, unclear, unknown |
| dis- | 否定、相反 | disagree, dislike, disconnect |
| pre- | 提前、预先 | prepare, preview, preheat |
| mis- | 错误地 | misunderstand, mistake |
| over- | 过度、超过 | overwork, overload |
| under- | 不足、低于 | underestimate, underpay |
| in- | 不、进入 | incorrect, input |
| im- | 不 | impossible, impatient |
| il- | 不 | illegal, illogical |
| ir- | 不 | irregular, irresponsible |
| inter- | 在……之间 | international, interact |
| intra- | 在……内部 | intranet |
| trans- | 跨越、转换 | transport, translate |
| super- | 超级、超过 | supermarket, superhuman |
| sub- | 在下面、子级 | subway, subtitle |
| auto- | 自动、自身 | automatic, autopilot |
| multi- | 多个 | multimedia, multitask |
| mono- | 单个 | monologue |
| bi- | 两个 | bicycle, bilingual |
| tri- | 三个 | triangle |
| co- | 共同、一起 | cooperate, coworker |

```text
-er    谁做（人）
-tion  做什么（过程）
-ment  做完了（结果）
-able  能不能做

```
