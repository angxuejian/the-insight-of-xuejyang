# be 家族

| [原型](#原型) | [现在时](#现在时) | [过去时](#过去时) | [过去分词](#过去分词) | [现在分词](#现在分词) |
| ------------- | ----------------- | ----------------- | --------------------- | --------------------- |
| be            | am / is / are     | was / were        | been                  | being                 |


### 原型

#### `be`

> 凡是英语语法要求"动词原形"的时候，be 就会出现。

| 结构            | 为什么是 be？   |
| --------------- | --------------- |
| to **be**       | to 后接原形     |
| will **be**     | will 后接原形   |
| can **be**      | can 后接原形    |
| should **be**   | should 后接原形 |
| must **be**     | must 后接原形   |
| let him **be**  | let 后接原形    |
| **Be** careful! | 祈使句用原形    |

### 现在时

#### `am`

> 只搭配 `I` 使用
>
> 举例：I `am` happy.

#### `is`

> 第三人称单数
>
> He `is` busy.
>
> She is beautiful.
>
> It is raining.
>
> GPT is down.

#### `are`

> 其它复数或者 you
>
> You are right.
>
> We are ready.
>
> They are students.

### 过去时

#### `was`

> 单数
>
> I was tired.
>
> He was at home.

#### `were`

> 复数
>
> They were late.
>
> We were happy.

### 过去分词

#### `been`

> I have been there.
>
> I've been busy lately.

### 现在分词

#### `being`

> 正在处于……
>
> Stop being rude. （不要一直处于 rude 的状态。）
>
> He is being strange. （目前处于这种状态）

## 含义

### 1、处于某种状态

```bash
# 举例
I am happy.

# 不要翻译
我是开心。

# 而是理解
我处于开心这个状态。
```

### 2、处于某个地方

```bash
# 举例
He is in Beijing.

# 不要翻译
他是在北京。

# 而是
他处于北京。/ 他人在北京。
```

### 3、某物存在

```bash
# 举例
There is a book.

# 不是翻译
那里是一本书。

# 而是
有一本书存在。
```

### 4、表示身份

```bash
# 举例
He is a doctor.

# 可以理解
他处于医生这个身份。
他的身份是医生。
```

### 共同点

```bash
I am happy.
        ↓
    存在于快乐状态

He is at home.
        ↓
    存在于家里

She is a teacher.
        ↓
    存在于教师身份

The meeting is tomorrow.
        ↓
    存在于明天这个时间

There is a book.
        ↓
    一本书存在
```

其实全都是：**存在**

所以 be 描述的是：**状态、位置、身份、存在**

> 某个人、某个东西，存在于某种状态、位置、身份或时间中。

> 主语 + be + 状态 / 身份 / 位置 / 存在

#### 举例

看到 be（am / is / are）时，只问自己一个问题，后面描述的是什么？

如果后面是：

- happy → 状态
- busy → 状态
- down → 状态
- online → 状态
- ready → 状态
- at home → 位置
- in China → 位置
- a teacher → 身份
- tomorrow → 时间（某些表达中）
