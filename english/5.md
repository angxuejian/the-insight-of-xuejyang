# 语法形变

> 为了满足语法规则，同一个单词改变形式，但不创造新的单词。

| 类型      | 示例                              | 属于语法形变（Inflection） | 说明          |
| ------- | ------------------------------- | ------------------ | ----------- |
| 名词      | book → books                    | ✅                  | 单复数变化       |
| 动词      | walk → walks / walked / walking | ✅                  | 三单、过去式、现在分词 |
| 形容词     | big → bigger → biggest          | ✅                  | 比较级、最高级     |
| 代词      | I → me / my / mine              | ✅                  | 格（case）的变化  |
| 动词不规则变化 | go → went → gone                | ✅                  | 不规则的时态/分词变化 |
