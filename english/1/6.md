# 常见英语介词（Prepositions）

| 介词 | 核心感觉 | 常见搭配 | 示例 |
|---|---|---|---|
| in | 在里面 | in the room / in Vue / in 2025 | He is in the room. |
| on | 在表面 | on the table / on GitHub | The book is on the table. |
| at | 在某点 | at school / at work / at 8:00 | She is at work. |
| to | 朝向 | go to / talk to | I go to school. |
| for | 为了 / 给 | for users / for testing | This tool is for users. |
| with | 和 / 使用 | with friends / with Python | I work with Python. |
| by | 通过 / 在旁边 | by car / by default | We test it by script. |
| from | 来自 | from China / from API | Data comes from the API. |
| about | 关于 | about Vue / about issues | We talked about Vue. |
| of | 的 / 属于 | part of / name of | This is part of the system. |
| after | 在之后 | after class / after work | We met after work. |
| before | 在之前 | before dinner / before deploy | Check it before deploy. |
| during | 在期间 | during testing / during class | Errors happened during testing. |
| through | 穿过 / 通过 | through tunnel / through API | Requests go through the gateway. |
| between | 在之间 | between A and B | The server sits between clients and APIs. |

# 快速识别技巧

看到下面这种结构：

- in + 名词
- on + 名词
- at + 名词
- for + 名词
- with + 名词
- by + 名词

大概率就是：

> 介词短语（不是句子主干）

- in the project
- for customers
- with Vue
- by default
- during testing

它们通常只是补充说明：
- 时间
- 地点
- 方式
- 原因
- 对象
