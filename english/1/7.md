# 限定词（Determiners）

| 类型                    | 常见词                                               | 核心作用         | 示例                |
| --------------------- | ------------------------------------------------- | ------------ | ----------------- |
| 冠词（Articles）          | a / an / the                                      | 指定“一个”还是“那个” | a book / the book |
| 指示限定词（Demonstratives） | this / that / these / those                       | 指“这个/那个”     | this issue        |
| 数量限定词（Quantifiers）    | some / many / much / few / little / several / all | 表示数量         | many engineers    |
| 所有格限定词（Possessives）   | my / your / his / her / our / their               | 表示所属关系       | our project       |
| 疑问限定词（Interrogatives） | which / what / whose                              | 用于提问         | which tool        |
| 分配限定词（Distributives）  | each / every / either / neither                   | 强调个体分配       | each component    |
| 数词限定词（Numerals）       | one / two / first / second                        | 表示数字或顺序      | first version     |
| 不定限定词（Indefinite）     | another / other / any / no                        | 泛指、不确定对象     | another engineer  |
| 全称限定词（Universal）      | all / both / whole                                | 表示整体范围       | all features      |
