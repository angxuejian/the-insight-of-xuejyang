# 时间、视角、扩展


## 时态（时间）

| 维度 | 作用           |
| -- | ------------ |
| 时间 | 过去 / 现在 / 将来 |
| 状态 | 一般 / 进行 / 完成 |

| 时间 | 一般    | 进行    | 完成    |
| -- | ----- | ----- | ----- |
| 现在 | 一般现在时 | 现在进行时 | 现在完成时 |
| 过去 | 一般过去时 | 过去进行时 | 过去完成时 |
| 将来 | 一般将来时 | 将来进行时 | 将来完成时 |

| 时态   | 公式              | 核心感觉    |
| ---- | --------------- | ------- |
| 一般现在 | do/does         | 经常      |
| 现在进行 | am/is/are doing | 正在      |
| 现在完成 | have/has done   | 已经      |
| 一般过去 | did             | 过去      |
| 过去进行 | was/were doing  | 当时正在    |
| 过去完成 | had done        | 过去之前已完成 |
| 一般将来 | will do         | 将来      |
| 将来进行 | will be doing   | 将来正在    |
| 将来完成 | will have done  | 将来已完成   |

## 语态（视角）

### 主动语态（Active Voice）

主语自己做动作。-> I eat an apple.

### 被动语态（Passive Voice）

主语是动作的承受者。-> The apple is eaten by me.

### 时态 × 语态

| 时态   | 主动                   | 被动                         |
| ---- | -------------------- | -------------------------- |
| 一般现在 | I write.             | It is written.             |
| 一般过去 | I wrote.             | It was written.            |
| 一般将来 | I will write.        | It will be written.        |
| 现在进行 | I am writing.        | It is being written.       |
| 过去进行 | I was writing.       | It was being written.      |
| 现在完成 | I have written.      | It has been written.       |
| 过去完成 | I had written.       | It had been written.       |
| 将来完成 | I will have written. | It will have been written. |

## 从句（扩展）

| 从句类型  | 作用    | 相当于 |
| ----- | ----- | --- |
| 名词性从句 | 当名词用  | 名词  |
| 定语从句  | 当形容词用 | 形容词 |
| 状语从句  | 当副词用  | 副词  |

### 名词性从句
可以做 主语从句、宾语从句、表语从句。

常见引导词：

| 引导词     | 含义    |
| ------- | ----- |
| that    | 无实际意义 |
| whether | 是否    |
| if      | 是否    |
| what    | 什么    |
| who     | 谁     |
| why     | 为什么   |
| how     | 如何    |

主语从句：`What he said` is true.

宾语从句：I know `that he is a teacher`.

表语从句：The problem is `The problem is that we are late`.

### 定语从句

定语 = 修饰名词。| 定语从句 = 用一句话修饰名词。

常见关系词：

| 关系词   | 指代  |
| ----- | --- |
| who   | 人   |
| whom  | 人   |
| whose | 谁的  |
| that  | 人/物 |
| which | 物   |

The man `who is standing there` is my teacher.

The `tall` man is my teacher.

I bought a book `that is very interesting`.


### 状语从句

状语 = 说明`时间`，`原因`，`条件`，`结果`，`让步`

常见连接词：

| 类型 | 连接词                        |
| -- | -------------------------- |
| 时间 | when, while, before, after |
| 原因 | because, since             |
| 条件 | if, unless                 |
| 让步 | although, though           |
| 结果 | so that                    |
| 目的 | in order that              |

时间状语从句：`When I arrived`, he was sleeping.

原因状语从句：`Because it rained`, we stayed home.

条件状语从句： `If you study hard`, you will pass.

让步状语从句：`Although he is rich`, he is unhappy.
